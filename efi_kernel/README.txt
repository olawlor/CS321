This is a 64-bit EFI boot program, which runs at startup on 
a 64-bit x86 PC with UEFI boot enabled.

To run:
	make run


Installation requirements:
	sudo apt-get install g++ build-essentials gnu-efi  mtools  qemu-system-x86 ovmf
